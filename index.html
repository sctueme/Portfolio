Independent Project Portfolio

Santiago Cabello Tueme

Airbnb Analytics

Text description about properties in Airbnb data across different cities allows us to form relevant clusters via PCA+Gaussian Mixtures to gain insight about the characteristics and value of those properties, easily readable with their associated word-clouds.

Wearable Device Data for Predicting Physical Activity States

A successful attempt to replicate (and in some aspects outperform) the results of Fuller et al's research paper. The aim is to classify the level of physical activity (or activity state) of participants with measurements taken from Apple Watch & Fitbit data, proving that these devices can be useful for developing other health-related apps that work based on their built-in measurement-capabilites.

Beer Style Similarity

Hierarchical clustering of beerstyles using data of thousands of home-brewing recipes. Dendrogram shows similarities between different styles and label color is asociated to beer color.

Trust-Region Optimization

Trust-Region numerical optimization methods using the Cauchy and Dog-leg points, for my Applied Analysis course. These methods are implemented from scratch using Matlab.

Airbnb Analytics

Numerical information in Airbnb data is not detailed enough for strong predictive analytics. However, we can take advantage of the available information in the text description of properties in different cities to form relevant groups via PCA+Gaussian Mixtures to gain a little more insight about the available information, easily readable with their associated word-clouds.
The code is available here.

Wearable Device Data for Predicting Physical Activity States

A successful attempt to replicate (and in some aspects outperform) the results of Fuller et al's research paper. The aim is to classify the level of physical activity (or activity state) of participants with measurements taken from Apple Watch & Fitbit data, proving that these devices can be useful for developing other health-related apps that work based on their built-in measurement-capabilites.


A summary of the project is available here. The code is available here.

Complexity vs Insufficient Data

An experiment to show how model complexity does not compensate for the lack of important data. The task is classifying beer styles based on very general information about recipe samples. A dendrogram is built as a proxy for similarity between styles.

A summary of the project is available here. The code is available here.

Trust-Region Optimization

Trust-Region numerical optimization methods using the Cauchy and Dog-leg points, for my Applied Analysis course. These methods are implemented from scratch using Matlab.

The code is available here.

Interior-point Optimization Method

Implementation and performance test of an Interior-points method with an analytically-reduced linear system for quasi-Newton matrix updates (for Hessian approximation). This was a team effort for a course in Numerical Optimization. Our implementation performs almost as well as Matlab's fmincon method.

The code is available here.

Solving Partial Differential Equations

Numerical implementation of finite difference methods (Crank-Nicolson, Lax-Friedrichs, and Leap-frog) for solving heat and transport differential equations, for my Numerical Analysis course.
